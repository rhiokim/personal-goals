# CTO CLI
CTO CLI 는 기술 영역에서 필요한 codeless 를 구현합니다.

## Basic

```bash
cto repo new-project
```

## Scope
* Repository
  * auto label templating
  * conventional commit lint
  * syncronize label
* CD
* CI

## Subcommands

```bash
cto repo
```
